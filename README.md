# GitBurger 

GitBurger is a group project developed as part of our Software Engineering studies.  
The project simulates a simple restaurant ordering system where orders are placed and handled through different views.

This was our first programming project, and the main focus was to learn how a web application is structured, how different parts communicate and how to work together in a development team.

##  Running the Project

The application is designed to be run using **Docker Compose**.

Requirements:
- Docker
- Docker Compose

Start the application from the project root:
```bash
docker compose up
```
